# Portfolio
## Korte uitleg
Deze repository bevat alle files voor mijn portfolio website die je via [deze link](https://www.milanvanwinkel.be) kunt bereiken. Op deze website kun je allerlei informatie over mij terugvinden, waaronder studies, ervaringen, skills... In deze README wil ik graag een duidelijker beeld geven over hoe het project precies in elkaar zit en welke technologieën ik zo al gebruik.

## Wat is dit project?
Dit project is mijn eigen portfolio website, die ik destijds heb moeten maken voor mijn jurymoment in juni 2025. Dit portfolio is gemaakt via het Laravel-framework, waar ik de HTML-code op deel in verschillende "views" met behulp van PHP en Blade. 

## Waarom heb ik dit project gemaakt?
Ik had al wel eens een portfolio site gemaakt in het verleden, helemaal in het begin van mijn opleiding, maar deze was enkel aan de hand van pure HTML, CSS en Bootstrap gemaakt. Na het afronden van mijn werkplekleren in mei 2025, moest ik voor het jury moment in juni 2025, een professionele website maken waar ik mijn ervaringen en projecten op moest weergeven. Ik was initieel gestart vanuit het niets, maar na enkele dagen heb ik toch besloten om vanuit een template te werken die ik online had gevonden. Achteraf gezien ben ik tevreden dat ik deze keuze heb gemaakt aangezien het een goede indruk heeft gegeven aan de juryleden tijdens mijn jurymoment. 

## Wanneer ben ik begonnen aan dit project?
Ik ben deze site gestart eind mei 2025 en wordt tot op heden nog altijd geupdatet.
